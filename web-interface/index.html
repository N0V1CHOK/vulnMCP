🚀 Quick Start Guide

  1. Install & Run:
    git clone https://github.com/YOUR_ORG/vulnmcp.git
    cd vulnmcp
    docker-compose up -d
  2. Configure Claude Desktop:

    Add to your claude_desktop_config.json:

    {
    "mcpServers": {
    "vulnmcp": {
    "command": "docker",
    "args": ["exec", "-i", "vulnmcp-server", "python", "-m", "src.server"]
    }
    }
    }
  3. Start Hacking:

    In Claude, use: vulnmcp_help to begin!
