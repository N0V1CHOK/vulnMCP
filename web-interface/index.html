VulnMCP

MCP Security Training Platform

Warning: This is an intentionally vulnerable environment for educational purposes only. Do not deploy in production or expose to the internet.

Setup

Get started in 3 steps:

  1. Clone and start the server 
    git clone https://github.com/N0V1CHOK/vulnMCP.git
cd vulnMCP
docker-compose up -d
  2. Configure Claude Desktop

    Add this to your claude_desktop_config.json file:

    {
  "mcpServers": {
    "vulnmcp": {
      "command": "docker",
      "args": ["exec", "-i", "vulnmcp-server", "python", "-m", "src.server"]
    }
  }
}
  3. Start training

    Open Claude Desktop and type: vulnmcp_help

What's Included

  • 8 security challenges covering common MCP vulnerabilities
  • Point-based scoring system with leaderboard
  • Built-in hint system (costs points)
  • Detailed explanations for each vulnerability

Commands

Available commands in Claude:

  • vulnmcp_help - Show help and available commands
  • vulnmcp_challenges - List all challenges
  • vulnmcp_start [level] - Start a specific challenge
  • vulnmcp_hint - Get a hint for current challenge
  • vulnmcp_submit [flag] - Submit a flag
  • vulnmcp_leaderboard - View the leaderboard

Requirements

  • Docker and Docker Compose
  • Claude Desktop (for MCP integration)
  • Basic understanding of security concepts

Troubleshooting

If you run into issues:

  • Check that Docker containers are running: docker-compose ps
  • View logs: docker-compose logs -f
  • Restart containers: docker-compose restart
  • Make sure ports 8080 and others aren't in use
